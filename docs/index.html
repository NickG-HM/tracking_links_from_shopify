Quick Start

git clone https://github.com/NickG-HM/tracking_links_from_shopify.git
cd tracking_links_from_shopify
npm install

Create a .env file:

SHOPIFY_STORE_DOMAIN=your-store.myshopify.com
SHOPIFY_ADMIN_ACCESS_TOKEN=shpat_xxxxxxxxxxxx
PORT=3000

Run:

npm start

POST /api/links

  "trackingNumber": "1Z999AA10123456784",

Supported Carriers

  • USPS, UPS, FedEx
  • DHL (Express & eCommerce)
  • Canada Post, Royal Mail, Australia Post
  • YunExpress, 4PX, Yanwen, Cainiao
  • PostNL, Deutsche Post, Hermes/Evri, GLS, TNT
  • Amazon Logistics, SF Express

Unknown carriers use Parcels App for universal tracking.
